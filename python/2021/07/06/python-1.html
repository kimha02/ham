객체?

  • 본질적으로는 객체는 메모리 조각이다.
  • 파이썬에서는 모든것이 객체이다. (값, 연산, 함수, 클래스, 컴파일된 코드)
    • 예를들면 숫자(99)도 객체이며, 파이썬이 제공하는 연산들(더하기, 빼기)도 객체이다.
  • 내장객체의 타입을 알아보자.
